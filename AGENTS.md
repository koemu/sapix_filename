# AGENTS.md

## プロジェクト概要

- このプロジェクトは、SAPIX から受け取ったテキストのファイル名を自動命名するためのプロジェクトです。

## アーキテクチャ

- Python の現時点での最新バージョンを利用します。
- コマンドラインアプリケーションとして動作します。これは、ターミナルから実行できるアプリケーションです。できるかぎりシンプルに動作させるためです。

## コーディング規約

- PEP8 を基本とします。

## セットアップ

- pyenv を活用した環境構築を実施します。必ず実施してください。
- プロジェクトの慣行管理は venv を利用します。必ず実施してください。
- メソッドを1つ作った時に、必ずテストコードも作成します。

## 禁止事項

- .env.example を編集しない
- .env.local を編集しない
- .env を編集しない
- AGENTS.md を編集しない
- PLAN.md を編集しない

## 権限

### 自由にやっていいこと

- 作業ブランチの作成
- .git リポジトリそのものの作成
- ライブラリの追加
- ライブラリの削除　(むしろ不要になったものは積極的に削除すること)
- コミットメッセージの作成

### 確認してから行うこと

- すべてののコミットの実施
- Python のバージョンアップ

### 絶対にやらない

- master ブランチへの直接のコミット・プッシュ
- 全てのブランチに対するプッシュ
